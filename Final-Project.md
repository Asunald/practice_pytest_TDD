# Final Project

## 🔬 Part II — “Behind the Hook” Deep Technical Implementation (25%)

Beyond using existing AI tools, you must **deeply implement one selected project at the code level**.

This component is worth **25 points**.

---

## 🎯 Core Requirements

* Choose **one of the five projects**
* Re-implement or reconstruct its core functionality
* Avoid relying solely on ready-made end-user AI tools
* Build a structured technical solution using, for example:

  * Python
  * Machine learning and/or deep learning
  * Lower-level Cloud/Local APIs
  * Cloud GPUs (if needed)
  * Custom pipelines
  * Model fine-tuning or inference design
  * System-level orchestration

The goal is to move from simply *using* AI systems to *building and understanding* them.

---

## 🚀 Extension & Creativity

You are strongly encouraged to go beyond simple reproduction.

You may:

* Add additional features
* Redesign the architecture
* Improve performance or efficiency
* Introduce experimental variations
* Build a full application around the core system
* Make it more robust, scalable, or elegant
* Extend the scope into a larger AI system

Ambition is welcome. Depth is valued.

---

## 🧠 Educational Objective

This component aims to:

> Understand the mechanism behind the tool.
> Go beyond “button-click AI.”
> Explore architecture, workflow, and model-level reasoning.

You should demonstrate that you understand:

* What happens behind the interface
* How the models are structured
* How data flows through the system
* Where design decisions influence outcomes

---

## 📊 Evaluation Criteria (25 Points)

This part evaluates:

* Technical depth
* Code quality and clarity
* Sound engineering structure
* Understanding of underlying principles
* Experimental validation (if applicable)
* System design coherence

Clear documentation and well-structured explanation will significantly strengthen your evaluation.

---

# 👥 Teamwork & Presentation

* All work is conducted in **fixed teams of two**
* Teams remain unchanged throughout the course
* Both members receive the same base project grade

However:

* Individual contribution will be considered
* Transparency is required

---

## 🎤 Final Presentation Requirements

During the final presentation, each team must:

* Explain tool choices and technical decisions
* Demonstrate the “Behind the Hook” implementation
* Provide an internal contribution statement including:

  * Who was responsible for which components
  * Time investment
  * Technical responsibilities
  * Major challenges encountered

Professional communication and technical clarity are part of the evaluation.
